# FAKE-STORE
A minimal frontend practice project built with React and TypeScript that fetches product data from [FakeStoreAPI](https://fakestoreapi.com/) and demonstrates core frontend concepts:

- Data fetching (with loading and error states)
- Global state management using [Zustand](https://zustand.docs.pmnd.rs/getting-started/introduction)
- Custom hooks & components
- Category filtering via dropdown
- Client-side pagination
- Shopping cart (add, remove, update quantity)
